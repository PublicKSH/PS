1차원 배열의 초기화
1. 1차원 배열의 모든 원소 값을 0으로 하고 싶은 경우
int arr[5] = {};
int arr[5] = {0};
int arr[5] = {0,};
이때 int arr[5] = {1,3,4,} 를하면 배열은 1 3 4 0 0으로 초기화 된다.

2. 1차원 배열의 모든 원소 값을 n으로 하고 싶은경우
1) std::fill_n함수를 이용한다.
fill_nj(변경하려는 원소의 범위 시작주소, 변경하려는 원소 갯수, 변경 값)
이때 2번째 값부터 변경하고 싶으면 시작주소에 1을 더하면 된다.
2) std::fill 이용
fill(변경하려는 원소의 범위 시작주소, 변경하려는 원소의 범위 종료주소, 변경 값)
